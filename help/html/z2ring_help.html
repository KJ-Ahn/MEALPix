z2ring

Return HEALPix ring number for a given z

Contents

  • Syntax
  • Input Arguments
  • Return Arguments
  • Description
  • Example
  • See also

Syntax

zNum = z2ring(nSide,z)

Input Arguments

nSide      HEALPix resolution parameter
z          numeric array of z values

Return Arguments

nRing      size(z) numeric array of ring numbers

Description

HEALPix pixels centers are arranged in rings of constant z. Each ring spans a given range of z. z2ring returns the ring numbers that cover the input z values.

Example

nSide = 4;
zNum = z2ring(nSide,reshape((-7:7)*2/15,3,5))

zNum =

    14    11     9     6     4
    13    10     8     6     3
    12    10     7     5     2
