ang2vec

Convert from spherical to cartesian coordinates

Contents

  • Syntax
  • Input Arguments
  • Return Arguments
  • Example
  • See also

Syntax

xyz = ang2vec(tp)

Input Arguments

tp        cell of [2,1] angular locations (theta; phi)

Return Arguments

xyz       size(tp) celll array of [3,1] vectors

Example

tp = [acos(2*rand(1,12)-1); 2*pi*rand(1,12)];
tp = mat2cell(tp,2,ones(12,1));
tp = reshape(tp,3,4);
xyz = ang2vec(tp);
xyz{2,3}'
