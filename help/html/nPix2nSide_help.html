nPix2nSide

Finds number of HEALPix facet side subdivisions from number of pixels on sphere

Contents

  • Syntax
  • Input Arguments
  • Return Arguments
  • Example
  • See also

Syntax

nSide = nPix2nSide(nPix);

Input Arguments

nPix     number of pixels on sphere

Return Arguments

nSide    sub-divisions of HEALPix facet side

Example

nPix = 12*[1:4].^2;
nSide = nPix2nSide(nPix)

nSide =

     1     2     3     4
