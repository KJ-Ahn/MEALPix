nSide2res

Determine angular resolution of a HEALPix map

Contents

  • Syntax
  • Input Arguments
  • Return Arguments
  • Example
  • See also

Syntax

angres = nSide2res(nSide)

Input Arguments

nSide   HEALPix resolution parameter (may be an array)

Return Arguments

angres  Angular resolution in arcsec

Example

nSide = 2.^(8:12);
nSide2res(nSide)

ans =

  824.5167  412.2584  206.1292  103.0646   51.5323
